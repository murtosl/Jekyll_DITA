Write A Post

A post is a Markdown, HTML or other text-based file filled with site content. Posts are inserted into the layout by Jekyll when it creates the website.

Posts go into the _posts folder and can be a variety of file types. Markdown and HTML are the most common.

Jekyll can use a variety of file types, but the filename of a post must follow a specific format. This format indicates the date the post was made, and the topic of the post. The following format for the filename is required:

YEAR-MONTH-DAY-title
