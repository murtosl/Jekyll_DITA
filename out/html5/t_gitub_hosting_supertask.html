Host with Github Pages

GitHub Pages will host sites using Jekyll for free. This can be set up in 10-20 minutes.

This might not be included in the submitted project

Because we do this already.
  • Set Up Github.io Repo
    In order to display your project as a webpage, GitHub requires a [username].github.io repository be created.
  • Set Up Project Pages
    Project Pages can be created for any GitHub repository by storing a website in the gh-pages branch.
